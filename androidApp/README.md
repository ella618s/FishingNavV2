# ⚓ FishingNav V2 - 離線釣魚導航系統

**FishingNav V2** 是一款專為釣魚愛好者設計的 Android App。解決了海上網路信號不穩、陽光下螢幕辨識度低，以及定位圖標偏移等技術痛點。

## 🌟 技術亮點

### 1. 離線圖資自動化
* **自動初始化**：App 首次啟動會自動從 Assets 複製 MBTiles 地圖檔至系統路徑，解決用戶手動放置檔案的困擾。
* **離線優先**：系統優先讀取本地 `fishing_map.mbtiles` 檔案，確保在海上斷網時依然能精確導航。

### 2. 精準定位 UI (自定義 Canvas)
* **中心錨點校正**：針對 `osmdroid` 自定義圖標偏移問題，透過 `setDirectionAnchor(0.5f, 0.5f)` 與重繪三角形 Path，達成前進方向三角形與 GPS 圓圈完美對齊。
* **高對比視覺設計**：螢光綠三角形搭配黑邊陰影，確保在戶外高亮度環境下清晰可見。

### 3. 多模式切換與導航
* **衛星模式支援**：手動定義 Google 衛星圖源，解決內建圖源失效問題。
* **動態距離計算**：即時顯示目標與當前位置的直線距離。
